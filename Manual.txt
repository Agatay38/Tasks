1. Установите Python (если он не установлен):
Скачайте установочный файл Python с официального сайта Python.
Установите Python, следуя инструкциям. 
Для Windows OS: Во время установки обязательно выберите опцию "Add Python to PATH".

Для MacOS: 
Откройте терминал и введите команду:
python3 --version
Если Python 3 не установлен, скачайте его с официального сайта Python и установите.

2. Скопируйте файл "LogSys.py" в доступную папку, например на рабочий стол.
Тогда путь будет такой для MS Windows - C:\Users\ВашеИмяПользователя\Desktop
Для MacOS: Desktop


3. Откройте командную строку (cmd):
Windows: Нажмите клавиши Win + R, введите cmd и нажмите Enter.
MacOS: Вы можете найти терминал через Spotlight (нажмите Cmd + Пробел и введите Terminal) или через Программы -> Утилиты -> Терминал.

4. Перейдите в директорию с файлом:
Используйте команду cd, чтобы перейти в директорию, где находится ваш файл LogSys.py. 
Например, если файл находится на рабочем столе:
	Windows: cd C:\Users\ВашеИмяПользователя\Desktop
	MacOS: cd ~/Desktop
	
5. Запустите Python файл:
	Windows:
	- Введите команду для запуска файла:
		python LogSys.py
	- Если у вас установлен Python 3.x и он связан с командой python3, используйте:
		python3 LogSys.py
	
	MacOS:
	- Введите команду для запуска файла в sh:
		python3 LogSys.py
	- Если у вас установлен Python 3.x и он связан с командой python, используйте в sh:
		python LogSys.py

6. Ввод данных в консоли:
Следуйте инструкциям, отображаемым в консоли, чтобы вводить данные и взаимодействовать с системой управления учетными записями пользователей.
